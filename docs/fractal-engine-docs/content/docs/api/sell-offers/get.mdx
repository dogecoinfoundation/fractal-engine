# List Sell Offers

Returns a paginated list of sell offers with optional filtering by mint hash and offerer address.

## Endpoint

```
GET /sell-offers
```

## Query Parameters

- **limit** (optional): Maximum number of results per page (default: 100, max: 100)
- **page** (optional): Page number for pagination (default: 1)
- **mint_hash** (optional): Filter sell offers by specific mint hash
- **offerer_address** (optional): Filter sell offers by specific offerer address

### Error Responses

- **400 Bad Request**: Server error retrieving offers
- **500 Internal Server Error**: Server error

## Example Requests

### List all sell offers

```bash
curl "http://localhost:8080/sell-offers"
```

### List sell offers with pagination

```bash
curl "http://localhost:8080/sell-offers?page=2&limit=50"
```

### Filter by mint hash

```bash
curl "http://localhost:8080/sell-offers?mint_hash=abc123def456..."
```

### Filter by offerer address

```bash
curl "http://localhost:8080/sell-offers?offerer_address=D1A2B3C4D5E6F7..."
```

### Combined filters

```bash
curl "http://localhost:8080/sell-offers?mint_hash=abc123def456...&offerer_address=D1A2B3C4D5E6F7...&page=1&limit=25"
```

## JavaScript Example

```javascript
// Get all sell offers
const response = await fetch('/sell-offers');
const data = await response.json();

console.log(`Found ${data.total} sell offers`);
data.offers.forEach(item => {
  const { offer, mint } = item;
  console.log(`${offer.quantity} ${mint.title} tokens for ${offer.price} satoshis`);
});

// Get sell offers for specific mint
const mintResponse = await fetch('/sell-offers?mint_hash=abc123def456...');
const mintData = await mintResponse.json();
```

## Python Example

```python
import requests

# Get all sell offers
response = requests.get('http://localhost:8080/sell-offers')
data = response.json()

print(f"Found {data['total']} sell offers")
for item in data['offers']:
    offer = item['offer']
    mint = item['mint']
    print(f"{offer['quantity']} {mint['title']} tokens for {offer['price']} satoshis")

# Get sell offers with filters
filtered_response = requests.get('http://localhost:8080/sell-offers', params={
    'mint_hash': 'abc123def456...',
    'page': 1,
    'limit': 50
})
filtered_data = filtered_response.json()
```

## Response Fields

- **offers**: Array of sell offer objects with associated mint information
- **total**: Total number of sell offers matching the query
- **page**: Current page number
- **limit**: Number of results per page

### Offer Object Fields

- **id**: Unique sell offer identifier
- **hash**: Unique hash of the sell offer
- **offerer_address**: Address of the token seller
- **mint_hash**: Hash of the associated mint
- **quantity**: Number of tokens being sold
- **price**: Total asking price in satoshis
- **public_key**: Public key used for signature verification
- **signature**: Cryptographic signature of the offer
- **created_at**: Timestamp when the offer was created

### Mint Object Fields

- **hash**: Unique hash of the mint
- **title**: Title of the tokenized asset
- **description**: Description of the tokenized asset
- **fraction_count**: Total number of fractions/tokens
- **creator_address**: Address of the mint creator
- **feed_url**: URL to the asset's data feed
- **tags**: Array of tags associated with the mint
- **metadata**: Additional metadata object
- **requirements**: Requirements for token ownership
- **lockup_options**: Token lockup configuration
- **created_at**: Timestamp when the mint was created

## Pagination

The API uses offset-based pagination:
- Results are returned in pages with configurable size
- Use `page` and `limit` parameters to navigate through results
- Maximum limit per page is 100 to ensure good performance

## Notes

- Each sell offer includes both the offer details and associated mint information
- Sell offers represent tokens available for purchase
- Use this endpoint to browse available tokens in the marketplace
