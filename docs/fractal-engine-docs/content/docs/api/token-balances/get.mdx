# Get Token Balances

Returns the token balances for a specific address, optionally filtered by mint hash.

## Endpoint

```
GET /token-balances
```

## Query Parameters

- **address** (optional): Dogecoin address to get token balances for
- **mint_hash** (optional): Filter balances by specific mint hash

## Response

### Success (200 OK)

```json
[
  {
    "mint_hash": "abc123def456...",
    "balance": 150
  },
  {
    "mint_hash": "def789ghi012...",
    "balance": 75
  }
]
```

### Error Responses

- **500 Internal Server Error**: Failed to retrieve token balances

## Example Requests

### Get all token balances for an address

```bash
curl "http://localhost:8080/token-balances?address=D1A2B3C4D5E6F7..."
```

### Get balance for specific mint and address

```bash
curl "http://localhost:8080/token-balances?address=D1A2B3C4D5E6F7...&mint_hash=abc123def456..."
```

### Get all balances for a specific mint (across all addresses)

```bash
curl "http://localhost:8080/token-balances?mint_hash=abc123def456..."
```

## JavaScript Example

```javascript
// Get all token balances for an address
const response = await fetch('/token-balances?address=D1A2B3C4D5E6F7...');
const balances = await response.json();

balances.forEach(balance => {
  console.log(`Mint ${balance.mint_hash}: ${balance.balance} tokens`);
});

// Get balance for specific mint
const mintResponse = await fetch('/token-balances?address=D1A2B3C4D5E6F7...&mint_hash=abc123def456...');
const mintBalances = await mintResponse.json();
```

## Python Example

```python
import requests

# Get all token balances for an address
response = requests.get('http://localhost:8080/token-balances', params={
    'address': 'D1A2B3C4D5E6F7...'
})
balances = response.json()

for balance in balances:
    print(f"Mint {balance['mint_hash']}: {balance['balance']} tokens")

# Get balance for specific mint and address
mint_response = requests.get('http://localhost:8080/token-balances', params={
    'address': 'D1A2B3C4D5E6F7...',
    'mint_hash': 'abc123def456...'
})
mint_balances = mint_response.json()
```

## Response Fields

The response is an array of balance objects, each containing:

- **mint_hash**: Unique hash identifier of the mint/token
- **balance**: Number of tokens owned by the address for this mint

## Use Cases

- **Portfolio View**: Get all tokens owned by a user
- **Specific Token Balance**: Check balance for a particular token
- **Token Distribution**: See how tokens are distributed across addresses (when using mint_hash filter only)
- **Trading Validation**: Verify sufficient balance before creating sell offers

## Notes

- Balances reflect confirmed token ownership on the blockchain
- Both parameters are optional - you can query by address, mint, or both
- If no parameters are provided, all token balances across all addresses and mints are returned
- Balances are updated as token transfers are confirmed on the Dogecoin blockchain
