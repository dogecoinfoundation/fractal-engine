# List Unspent Outputs

Returns the unspent transaction outputs (UTXOs) for a given Dogecoin address.

## Endpoint

```
GET /list-unspent
```

## Query Parameters

- **address** (required): Dogecoin address to list unspent outputs for

## Response

### Success (200 OK)

```json
[
  {
    "txid": "abc123def456...",
    "vout": 0,
    "address": "D1A2B3C4D5E6F7...",
    "amount": 1000.0,
    "confirmations": 6,
    "scriptPubKey": "76a914..."
  },
  {
    "txid": "def789ghi012...",
    "vout": 1,
    "address": "D1A2B3C4D5E6F7...",
    "amount": 250.5,
    "confirmations": 3,
    "scriptPubKey": "76a914..."
  }
]
```

### Error Responses

- **400 Bad Request**: Address is required
- **500 Internal Server Error**: Error retrieving unspent outputs

## Example

### cURL

```bash
curl "http://localhost:8080/list-unspent?address=D1A2B3C4D5E6F7..."
```

### JavaScript

```javascript
const address = 'D1A2B3C4D5E6F7...';
const response = await fetch(`/list-unspent?address=${address}`);
const utxos = await response.json();

let totalBalance = 0;
utxos.forEach(utxo => {
  totalBalance += utxo.amount;
  console.log(`UTXO: ${utxo.txid}:${utxo.vout} - ${utxo.amount} DOGE (${utxo.confirmations} confirmations)`);
});

console.log(`Total balance: ${totalBalance} DOGE`);
```

### Python

```python
import requests

address = 'D1A2B3C4D5E6F7...'
response = requests.get('http://localhost:8080/list-unspent', params={
    'address': address
})
utxos = response.json()

total_balance = sum(utxo['amount'] for utxo in utxos)
print(f"Found {len(utxos)} unspent outputs")
print(f"Total balance: {total_balance} DOGE")

for utxo in utxos:
    print(f"UTXO: {utxo['txid']}:{utxo['vout']} - {utxo['amount']} DOGE ({utxo['confirmations']} confirmations)")
```

## Response Fields

Each unspent output object contains:

- **txid**: Transaction ID containing the unspent output
- **vout**: Output index within the transaction
- **address**: The address that owns this output
- **amount**: Amount of DOGE in this output
- **confirmations**: Number of confirmations for this transaction
- **scriptPubKey**: The script that must be satisfied to spend this output

## Use Cases

- **Balance Calculation**: Sum all UTXOs to get total spendable balance
- **Transaction Building**: Select UTXOs as inputs for new transactions
- **Wallet Management**: Track available funds and confirmations
- **Fee Estimation**: Calculate transaction fees based on input count
- **Mint Preparation**: Verify sufficient funds before creating mints

## Notes

- Only returns UTXOs that are spendable (typically with at least 1 confirmation)
- UTXOs with more confirmations are generally preferred for important transactions
- The sum of all UTXO amounts equals the total spendable balance for the address
- This endpoint queries the connected Dogecoin node for real-time UTXO data
- UTXOs are consumed when creating transactions and new ones are created as outputs
