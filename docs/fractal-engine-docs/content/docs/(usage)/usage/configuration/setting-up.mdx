---
title: Setting Up A Node
---

By default, there are a few moving parts that you need running:

- Fractal Engine
- A PostgreSQL database
- Dogecoin Core, with a synced copy of the blockchain, and available **RPC & ZMQ** endpoints

## Dogecoin Core
See https://github.com/dogecoin/dogecoin/blob/master/INSTALL.md

## PostgresQL Database
See https://www.postgresql.org/download/

## Fractal Engine

### Getting the source
https://github.com/dogecoinfoundation/fractal-engine

### Running a simple node
```sh
export DATABASE_URL=<POSTGRES_INSTANCE_CONNECTION_URL>
export DOGE_SCHEME=http
export DOGE_HOST=<HOSTNAME_OF_YOUR_DOGECOIN_CORE_INSTALL>
export DOGE_PORT=<PORT_OF_YOUR_DOGECOIN_CORE_INSTALL>
export DOGE_USER=<USERNAME_OF_YOUR_DOGECOIN_CORE_INSTALL>
export DOGE_PASSWORD=<PASSWORD_OF_YOUR_DOGECOIN_CORE_INSTALL>

go run cmd/fractal-engine/fractal_engine.go
```
