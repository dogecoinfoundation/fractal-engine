### Creating a basic Invoice
```json
{
  "public_key": "YOUR_PUBLIC_KEY",
  "signature": "SIGNATURE",
  "payload": {
    "payment_address": "YOUR_ADDRESS",
    "buyer_address": "BUYER_ADDRESS",
    "mint_hash": "MINT_HASH",
    "quantity": 10,
    "price": 20,
    "seller_address": "YOUR_ADDRESS"
  }
}
```
